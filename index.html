06업체 문의 사항

Question 01

2인치 다운라이트 — 디밍 가능 / 디밍 불가 두 가지 옵션 추천 부탁드립니다

전 공간에 동일한 2인치 LED 다운라이트로 통일하려고 합니다. 디밍 가능 제품과 디밍 불가(일반) 제품 두 가지를 비교해서 추천 부탁드립니다.

요구 조건
• 타공: 50~74mm (2인치)
• 색온도: 전주백색(주광색~주백색, 4000K~5700K 영역) 선호
• 플리커프리, 잔불방지 기능 포함 제품
• 슬림 베젤 (천장 마감과 자연스럽게 어울리도록)
• 디밍 가능 제품: 디밍 방식(0-10V / PWM / 전용 디머 등) 함께 안내 부탁드립니다
• 거실 8개, 안방 4개 = 총 12개 (+ 추후 추가 공간 협의)
Question 02

냉장고장과 시스템에어컨 사이 — 다운라이트 설치 가능한지 확인 부탁드립니다

냉장고장과 시스템에어컨 사이의 천장 간격이 좁은데, 이 위치에 2인치 다운라이트를 설치할 수 있을지 현장에서 확인 부탁드립니다.

현장 사진 — 시스템에어컨(좌) 옆 냉장고장(우) 사이 천장 영역
확인 필요 사항
• 천장 매립 깊이 (50mm 이상 확보 가능 여부)
• 시스템에어컨 배관/덕트와의 간섭
• 냉장고장 상부 마감 라인과의 거리
• 타공 가능한 최소 폭
• 불가능할 경우 대체 위치 제안
